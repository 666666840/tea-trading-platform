# 数据安全与合规自查清单

## 1. 数据安全
- 是否对敏感数据（如手机号、身份证号）加密存储？
- 是否定期备份数据库？
- 是否有数据脱敏导出机制？
- 是否有日志留存和轮转机制？

## 2. 权限与操作安全
- 是否对所有敏感操作做了权限校验？
- 是否有操作日志和异常告警？
- 是否有二次确认机制？

## 3. 隐私合规
- 是否有用户隐私政策？
- 是否支持用户数据导出与删除？
- 是否有数据访问最小化原则？

## 4. 法律合规
- 是否关注并遵守相关法律法规（如网络安全法、数据安全法、GDPR等）？
- 是否定期进行合规自查和风险评估？

## 5. 其它
- 是否有应急预案和演练？
- 是否有安全培训和意识提升？
